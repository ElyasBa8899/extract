# سامانه مدیریت دبستان

این پروژه یک سامانه مدیریتی تحت وب است که با PHP و MySQL پیاده‌سازی شده و برای مدیریت فعالیت‌های داخلی یک سازمان آموزشی-فرهنگی طراحی شده است.

## قابلیت‌های کلیدی

- **مدیریت کاربران، نقش‌ها و دسترسی‌ها:** سیستم جامع برای تعریف نقش‌های مختلف و اعطای دسترسی‌های دقیق به هر نقش.
- **فرم‌ساز داینامیک:** قابلیت ایجاد فرم‌های دلخواه (مانند فرم خوداظهاری یا گزارش) توسط ادمین بدون نیاز به کدنویسی.
- **ماژول‌های تخصصی:** شامل بخش‌های نظارت، جذب و راه‌اندازی، ضمن خدمت، اولیا، پرورشی و مالی.
- **سیستم انبارداری و کرایه‌چی:** مدیریت اقلام موجود و ثبت تاریخچه کرایه آن‌ها.
- **سیستم مالی جزوات:** پیگیری حساب‌های مالی مدرسین مربوط به تحویل و پرداخت هزینه جزوات.
- **سیستم تیکتینگ:** ابزاری برای ارتباطات داخلی، ارجاع کار و ارسال پیام بین کاربران و بخش‌ها.
- **داشبورد هوشمند:** نمایش خلاصه‌ای از فعالیت‌ها، تیکت‌ها و وضعیت مالی کاربر پس از ورود.
- **سیستم نوتیفیکیشن:** اعلان‌های زنده برای اطلاع‌رسانی از رویدادهای مهم مانند پاسخ به تیکت.

## راهنمای نصب و راه‌اندازی

برای راه‌اندازی این پروژه به صورت লোকাল، مراحل زیر را دنبال کنید:

**پیش‌نیازها:**
- یک وب سرور محلی مانند **XAMPP** یا WAMP نصب شده باشد.

### مرحله ۱: کپی کردن فایل‌های پروژه

1.  تمام محتویات این پروژه را در یک پوشه به نام `dabestan` در پوشه `htdocs` محل نصب XAMPP خود کپی کنید.
    -   مسیر نهایی باید به این صورت باشد: `C:\xampp\htdocs\dabestan`

### مرحله ۲: ایجاد دیتابیس

1.  XAMPP را اجرا کرده و ماژول‌های Apache و MySQL را Start کنید.
2.  وارد **phpMyAdmin** شوید (معمولاً با آدرس `http://localhost/phpmyadmin`).
3.  یک دیتابیس جدید با نام `dabestan_db` بسازید. (Encoding را روی `utf8mb4_persian_ci` تنظیم کنید تا از زبان فارسی به درستی پشتیبانی شود).

### مرحله ۳: وارد کردن ساختار جداول (Import)

1.  دیتابیس `dabestan_db` را که در مرحله قبل ساختید، انتخاب کنید.
2.  به تب `Import` بروید.
3.  روی دکمه `Choose File` کلیک کرده و فایل `database.sql` موجود در ریشه پروژه را انتخاب کنید.
4.  روی دکمه `Go` در پایین صفحه کلیک کنید تا تمام جداول مورد نیاز در دیتابیس شما ایجاد شوند.

### مرحله ۴: اجرای اسکریپت راه‌اندازی اولیه

1.  مرورگر خود را باز کرده و آدرس زیر را وارد کنید:
    ```
    http://localhost/dabestan/setup.php
    ```
2.  این اسکریپت، دسترسی‌های اولیه سیستم را در دیتابیس ثبت کرده و کاربر ادمین اصلی را ایجاد می‌کند.
3.  پس از مشاهده پیام "Setup Complete"، **بلافاصله و حتماً فایل `setup.php` را از پوشه پروژه خود حذف کنید.** این کار برای حفظ امنیت سیستم ضروری است.

### مرحله ۵: ورود به سیستم

1.  به آدرس اصلی پروژه بروید:
    ```
    http://localhost/dabestan/
    ```
2.  با اطلاعات زیر وارد پنل مدیریت شوید:
    -   **نام کاربری:** `admin`
    -   **رمز عبور:** `Admin_dabestan_site_110_59`

اکنون سیستم آماده استفاده است. می‌توانید از پنل ادمین، کاربران و نقش‌های جدید تعریف کرده و کار با سامانه را آغاز کنید.
